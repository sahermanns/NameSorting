This is an example of an insertion sort algorithm. In an insertion sort the index[0] is considered sorted because it comes first and has nothing preceding it to be compared with. In the first interation of the sorting loop #89 stays where it is. The next index under review is index[1] whose value is #15, it is compared to #89 at index[0] and since #15 is less thn #89, Doug slides to index[1] and makes room for Jermaine in index[0]. This pattern continues until the final index value is greater than the one before it, then the sort stops. In my example you can sort by jersey# or by last name.
